Flexbox Grid Generator

Columnar is a tool I made for generating flexbox-based grids. Add or subtract columns, set gutter width, and drag the columns as wide or narrow as you’d like — then copy and paste the markup into your own project. The markup is actually quite simple, but it exposes how easy flexbox can be.
